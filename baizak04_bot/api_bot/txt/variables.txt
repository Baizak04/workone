variables - ПЕРЕМЕННЫЕ

В Python переменные используются для хранения данных и присваивания значений. Вот некоторые основные правила и примеры по работе с переменными в Python:

1. Создание переменной: Для создания переменной в Python необходимо выбрать имя переменной и присвоить ей значение с помощью оператора присваивания "=".
Пример:

python
Copy code
name = "John"  # Создание переменной name и присваивание значения "John"
age = 25      # Создание переменной age и присваивание значения 25

2. Типы данных: В Python переменные могут хранить различные типы данных, такие как строки (str), целые числа (int), числа с плавающей точкой (float), булевы значения (bool) и другие.
Пример:

python
Copy code
name = "John"  # Строковая переменная
age = 25      # Целочисленная переменная
pi = 3.14     # Переменная с плавающей точкой
is_student = True  # Булевская переменная

3. Изменение значения переменной: Значение переменной можно изменить путем повторного присваивания нового значения.
Пример:

python
Copy code
age = 25   # Значение переменной age равно 25
age = 30   # Изменение значения переменной age на 30
4. Имя переменной: Имя переменной может состоять из букв, цифр и символа подчеркивания (_). Оно должно начинаться с буквы или символа подчеркивания и не может быть ключевым словом Python.
Пример:

python
Copy code
my_variable = 10   # Правильное имя переменной
2nd_variable = 20  # Неправильное имя переменной (не может начинаться с цифры)
5. Присваивание значений нескольким переменным: В Python можно одновременно присвоить значения нескольким переменным.
Пример:

python
Copy code
x = y = z = 0  # Присваивание значения 0 трем переменным x, y и z
Python является языком с динамической типизацией, поэтому переменные не требуют объявления типа. Они могут автоматически изменять свой тип во время выполнения программы в зависимости от присваиваемых значений.
