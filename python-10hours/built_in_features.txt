В Python существует множество встроенных функций, которые можно использовать без необходимости импорта дополнительных модулей. Вот некоторые из наиболее распространенных встроенных функций Python:

print(): Используется для вывода информации на экран.

input(): Позволяет пользователю вводить данные с клавиатуры и возвращает введенное значение.

len(): Возвращает длину объекта, такую как строка, список или кортеж.

type(): Возвращает тип объекта.

int(), float(), str(): Преобразуют объекты в целые числа, числа с плавающей запятой или строки соответственно.

range(): Создает последовательность чисел.

max(), min(): Возвращают наибольший и наименьший элементы в последовательности соответственно.

sum(): Возвращает сумму всех элементов в последовательности.

abs(): Возвращает абсолютное значение числа.

round(): Округляет число до заданного количества знаков после запятой.

sorted(): Сортирует элементы последовательности.

zip(): Создает итератор, который объединяет элементы из нескольких последовательностей.

map(): Применяет заданную функцию ко всем элементам последовательности.

filter(): Фильтрует элементы последовательности с помощью заданного условия.

all(): Возвращает True, если все элементы последовательности истинны, и False в противном случае.

any(): Возвращает True, если хотя бы один элемент последовательности истинен, и False в противном случае.

Это только некоторые из множества встроенных функций, доступных в Python. Вы можете узнать больше о встроенных функциях, изучая документацию Python.
